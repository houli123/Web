# 旋转法则
已知：
宇宙universe
宇宙的本地坐标系是[O1;i1,j1,k1]
O1(0,0,0)
i1(1,0,0)
j1(0,1,0)
k1(0,0,1)
银河系galaxy
银河系的本地坐标系是[O2;i2,j2,k2]
O2(1,2,3)
i2(1,0,0)
j2(0,1,0)
k2(0,0,1)
太阳系的本地坐标系是[O3;i3,j3,k3]
O3(4,5,6)
i3(1,0,0)
j3(0,1,0)
k3(0,0,1)

让银河系的本地坐标系[O2;i2,j2,k2]绕j2轴逆时针旋转20°。
设：c2=cos(-20°)，s2=sin(-20°)
则：
O2(1,2,3)
i2(c2,0,-s2)
j2(0,1,0)
k2(s2,0,c2)

让太阳系的本地坐标系[O3;i3,j3,k3]绕k3轴逆时针旋转30°
设：c3=cos(30°)，s3=sin(30°)
则：
O3(4,5,6)
i3(c3,-s3,0)
j3(s3,c3,0)
k3(0,0,1)

求：地球的世界坐标位P1
解：
由银河系的本地坐标系可得矩阵m2：
```html
[
    c2, 0, s2, 0,
    0,  1, 0,  0,
    -s2,0, c2, 0,
    1,  2, 3,  1
]
```

由太阳系的本地坐标系可得矩阵m3：
```html
[
    c3,  s3, 0, 0,
    -s3, c3, 0, 0,
    0,   0,  1, 0,
    4,   5,  6, 1
]
```
求地球的世界坐标位P1：
```html
M = m1*m2*m3=[
c2*c3,      s3,   s2*c3,      0,
-c2*s3,     c3,   -s2*s3,     0,
-s2,        0,    c2,         0,
c2*4-s2*6+1,5+2,s2*4+c2*6+3,1
]

```
